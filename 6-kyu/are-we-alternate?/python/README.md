Create a function `isAlt()` that accepts a string as an argument and validates whether the vowels (a, e, i, o, u) and consonants are in alternate order.

```javascript
isAlt("amazon")
// true
isAlt("apple")
// false
isAlt("banana")
// true
```
```haskell
isAlt "amazon" -> True
isAlt "apple"  -> False
isAlt "banana" -> True
```
```py
is_alt("amazon")
// true
is_alt("apple")
// false
is_alt("banana")
// true
```
```C#
Kata.IsAlt("amazon")
// true
Kata.IsAlt("apple")
// false
Kata.IsAlt("banana")
// true
```
```coffeescript
isAlt 'amazon'
# true
isAlt 'apple'
# false
isAlt 'banana'
# true
```

Arguments consist of only lowercase letters.
